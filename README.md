# 🎖️ منظومة إدارة التدريب والترشيح العسكري | Military Training Management System

نظام ويب متكامل ومتطور ومبني بأحدث تقنيات الويب، مصمم خصيصاً لإدارة، أتمتة، ومتابعة عمليات الترشيح للدورات التدريبية الموجهة لمنسوبي القطاعات العسكرية والأمنية من (ضباط وأفراد). يوفر النظام لوحة تحكم ذكية تضمن مرونة عالية وسرعة فائقة في معالجة البيانات وإدارة الموارد البشرية.

## ✨ أبرز المميزات (Key Features)

* **إدارة تصنيفات الرتب:** فصل كامل ومرن في عرض وإدارة بيانات الضباط والأفراد بما يتناسب مع الهيكل الإداري.
* **تتبع الحالات الإدارية:** مؤشرات حية وديناميكية لحالة كل منسوب (نشط، غياب، تأخير، إجازة، إذن، مسلم).
* **محرك تقارير إكسيل ذكي:** تصدير كشوفات الترشيح الفورية بصيغة `.xls` متوافقة برمجياً مع حقول المنظومة والرموز العسكرية (مثل حقل B/N والأرقام العسكرية).
* **واجهة مستخدم عصرية (UI/UX):** تصميم لوحة تحكم داكنة (Dark Theme) بالكامل، متجاوبة ومريحة للعين، ومبنية لمنع التمرير الأفقي المزعج في الجداول الكبيرة.
* **إدارة المرفقات الآمنة:** نظام رفع وتخزين ذكي لصور المنسوبين الرسمية مع معالجة برمجية تلقائية للحالات الاستثنائية والصور المفقودة.

## 🚀 التقنيات المستخدمة (Tech Stack)

* **Backend:** Laravel 11 (PHP)
* **Frontend:** Livewire, Tailwind CSS, Alpine.js (TALL Stack)
* **Database:** MySQL
